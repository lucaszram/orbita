## Producto

Órbita parte de datos de nacimiento para construir una experiencia personalizada:

- Fecha de nacimiento: ubica el Sol y base astrologica.
- Lugar de nacimiento: ayuda a calcular ascendente y casas.
- Hora de nacimiento: afina ascendente y casas; si falta, se usa una carta aproximada.
- Carta natal/base: punto de partida para lecturas.
- Transitos del dia: contexto diario personalizado.
- Guía diaria: micro-rituales, insights y señales para este momento.
- Pago de onboarding: desbloqueo de carta, transitos y lecturas mas profundas.
